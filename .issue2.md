## Beschreibung

BIOMETRICS braucht eine native Go-CLI für:
- Schnelle Ausführung
- Bessere Integration mit Dev-Tools
- Enterprise-Features

## Anforderungen
- Go 1.21+ CLI App
- Commands: init, check, run, config
- Autocomplete support
- Cross-platform binaries

## Acceptance Criteria
- [ ] Go CLI App erstellt
- [ ] `biometrics init` command
- [ ] `biometrics check` command
- [ ] `biometrics --version` output
- [ ] cross-platform build script
