## Purpose

Command-line interface and automation for biometric data collection, validation, and reporting with enterprise-grade security and performance.

## Structure

```
biometrics-cli/
├── cmd/                    # Command entrypoints
├── commands/                # CLI command implementations
├── pkg/                    # Core packages
│   ├── audit/              # Audit logging
│   ├── performance/         # Performance profiling
│   └── validation/         # Input validation
├── global/                 # Global configuration
├── local/                  # Local configuration
├── templates/              # Report templates
├── docs/                   # Documentation
├── scripts/                # Build and utility scripts
├── Makefile                # Build automation
├── go.mod                  # Go module definition
├── go.sum                  # Dependency checksums
└── main.go                 # Application entry point
```

## Features

- **Input Validation**: Comprehensive request validation with sanitization
- **OAuth2 Integration**: Secure authentication flows
- **Audit Logging**: Full event tracking and storage
- **Performance Profiling**: CPU, memory, goroutine, and mutex profiling
- **Secure Storage**: Encrypted credential management
- **Report Generation**: Configurable output formats

## Installation

```bash
# Build from source
make build

# Run tests
make test

# Generate coverage report
make coverage
```

## Usage

```bash
# Display help
./biometrics --help

# Run specific command
./biometrics [command] [flags]
```
