# CODE_OF_CONDUCT.md

## Global-Mandate-Alignment (AGENTS-GLOBAL)

- Zusammenarbeit folgt globaler Verantwortungs- und Eskalationsdisziplin.
- Transparenz, Ownership und Nachvollziehbarkeit gelten als Verhaltensstandard.
- Regelverstöße werden dokumentiert und strukturiert adressiert.

Status: ACTIVE  
Version: 1.0 (Universal)  
Stand: Februar 2026

## Zweck
Universeller Verhaltensrahmen für professionelle, respektvolle und verlässliche Zusammenarbeit.

## Leitsätze
1. Respektvoll kommunizieren
2. Verantwortung übernehmen
3. Transparenz vor Schuldzuweisung
4. Faktenbasiert arbeiten

## Erwartetes Verhalten
- klare, sachliche Kommunikation
- konstruktives Feedback
- aktive Dokumentation von Entscheidungen
- frühzeitige Meldung von Risiken/Blockern

## Nicht toleriertes Verhalten
- absichtliche Täuschung
- respektloses Verhalten
- Verschweigen kritischer Risiken
- unautorisierter Umgang mit sensiblen Daten

## Meldemechanismus
1. Vorfall dokumentieren
2. zuständige Rolle informieren
3. Schweregrad klassifizieren
4. Maßnahme einleiten

## Konsequenzrahmen
- Hinweis
- temporäre Einschränkung
- Ausschluss aus kritischen Rollen

## Abnahme-Check CODE_OF_CONDUCT
1. Erwartetes Verhalten klar
2. Nicht toleriertes Verhalten klar
3. Meldemechanismus vorhanden
4. Konsequenzrahmen vorhanden
